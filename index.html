GitHub Pages Slide Deck

Slide Site

このスライドを編集して、GitHub Pages で公開できます。

Overview

このプロジェクトについて

  • ビルド不要の静的サイトです。
  • GitHub Pages の root 配信でそのまま動きます。
  • HTML を編集するだけでスライドを追加できます。

Editing

スライドの追加方法

index.html に <section class="slide"> を追加してください。 

<section class="slide" data-title="New">
  <h2>新しいスライド</h2>
  <p>本文を書きます。</p>
</section>

Publish

公開設定

  1. GitHub に push
  2. Settings -> Pages を開く
  3. Deploy from a branch / main / root を選択
